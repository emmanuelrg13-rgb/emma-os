Sincronización con Google Sheets

Conecta Emma OS a una planilla de Google Sheets usando Apps Script. La app mantiene tus datos locales, pero puede subir un respaldo global y restaurar el último respaldo desde Sheets.

Guía de configuración

1
Crea una Google Sheet
Nombre sugerido: Emma OS - Base de Datos.
2
Abre Apps Script
En la planilla: Extensiones → Apps Script.
3
Pega el script
Usa el botón “Copiar Apps Script”, reemplaza CAMBIA_ESTA_CLAVE por tu clave privada y guarda.
4
Implementa como Web App
Implementar → Nueva implementación → Aplicación web. Ejecutar como tú. Acceso: cualquier usuario con el enlace.
5
Pega URL + clave en Emma OS
Guarda configuración, prueba conexión y luego sube respaldo.

Nota honesta: esta integración usa JSONP para leer y POST silencioso para subir, porque GitHub Pages y Apps Script están en orígenes distintos. Mantén el Centro de respaldos manual como plan B.
