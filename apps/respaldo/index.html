Protocolo antes de subir una versión nueva

1
Exporta respaldo global
Guarda el archivo JSON en una carpeta segura, Drive o tu computador.
2
Actualiza GitHub Pages
Sube los archivos nuevos del ZIP al repositorio y haz commit.
3
Prueba en computador
Abre la portada y entra a cada módulo. Verifica que tus datos sigan ahí.
4
Prueba en teléfono
Abre la app instalada. Si el navegador quedó con versión antigua, cierra y reabre la app.
5
Importa sólo si hace falta
Si algo no aparece, importa el respaldo global desde esta misma pantalla.
